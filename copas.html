Helixs.tech – Halo sobat Helix, kali ini Helix ingin membagikan Source Code Sngine, Apa sih Sngine itu? Sngine adalah Platform Social Media yang dibuat menggunakan bahasa pemrograman PHP, Sngine adalah cara terbaik untuk membuat situs web sosial atau komunitas online Anda sendiri. Luncurkan hanya dalam 1 menit dengan fitur-fitur terbaik. Cepat, aman, dan akan diperbarui secara rutin. Nah utnk Fitur nya sangat banyak seperti dibawah ini.

User/General Features:

  • Friends Users dapat menambahkan orang lain sebagai teman
  • Follow/Unfollow Sngine juga mendukung sistem berikut seperti Twitter / Facebook
  • Real-Time Real-Time Umpan Berita, Obrolan, & Pembaruan Profil
  • Share Users dapat membagikan pos publik apa pun di Sngine
  • Smart Publisher Publisher auto-scrap (Music, Videos, Links)
  • YouTube, Vimeo, SoundCloud Users dapat membagikan tautan dari media ini
  • Notifications Dapatkan notifikasi dari pengguna lain saat mereka: Suka, Bagikan, Komentar, atau @mention Anda
  • Sound Notifications untuk Pemberitahuan & Pesan Baru
  • Privacy users dapat mengubah pengaturan privasi mereka kapan saja
  • #Hahtags Post #Hashtags Saat Memposting, Berkomentar dan dalam obrolan
  • @mention user dapat mention teman mereka
  • Photos mengunggah gambar dan menampilkannya sebagai galeri
  • Groups User dapat berhubungan dengan anggota grup dan berbagi cerita dengan mereka
  • Pages User sekarang dapat membuat halaman seperti Facebook
  • Smiles / Emoticons (smiles) di Pesan, Komentar dan Obrolan
  • Manage Blocks Users dapat memblokir pengguna lain dan mencegah mereka melihat profil, dan menghubungi anda
  • Awesome Profile Dengan Sampul dan Avatar untuk profil pengguna / halaman / grup
  • Verify Badge Sekarang profile dan Halaman mendukung verify badge seperti centang bir pada Instagram
  • Unified Search Box Pencarian langsung untuk #hashtag, posting, halaman, dan grup
  • See More option untuk memotong posting teks panjang seperti Facebook
  • Retina Display Ready Responsive Design (dioptimalkan untuk Andorid dan perangkat iOS lainnya)
  • Suggestions Saran teman (untuk pengguna baru), Halaman untuk disukai, & Grup untuk bergabung
  • Movie User dapat Streaming Film/Video yang di unggah Admin
  • Games User dapat memainkan Games yang di unggah Admin
  • Offers Ajang diskon
  • Jobs User dapat membuat atau mencari lowongan pekerjaan
  • Events Membuat Event untuk di ikuti user lain
  • Blogs Berguna untuk Blogger, User dapat membuat artikel dan membagikannya secara online
  • Marketplace Halaman yang berguna ntuk menjual atau membeli sesuatu yang di tambahkan user lain
  • Forums Membuat Forum diskusi dan membagikannya

Requirements:

  • Sngine requires Apache Server, PHP 5.5+ (PHP 7.1+ to use Twilio Services) and MySQL to run
  • Before install Sngine please make sure that PHP extensions (mysqli, mbstring, curl) & PHP library (GD) & PHP funcrions (allow_url_fopen() & mail()) & Apache extension (mod_rewrite) are enabled on your server
  • SSL certificate (only required if you enabled video chat)
